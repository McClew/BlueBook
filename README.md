# Field Kit

## Field Manual

The Field Manual serves as a comprehensive and practical reference guide for conducting Blue Team operations. It has been designed and structured around key responsibilities/roles within defensive cyber security.

The key sections of the field manual are listed below:

#### Security Analysis

Knwoledge articles for analysing security alerts and monitoring for TTPs.

#### Indcident Response

Guides on evidence aquisition, initial triaging and containment techniques.

#### Digital Forensics

Forensic techniques beyong initial triaging and investigation, with guides on how to acquire specific evidence.

#### Reverse Engineering

Informational articles and walkthrough guides for conducting reverse engineering and malware analysis.

#### Security Engineering

Detailed guides of how to protect systems against common vulnerabilities, common misconfigurations seen in professional environments and other preventative measures.
