# Log Analysis

* Windows Event ID "Cheat Sheet": A table categorized by activity:
  * Logon: 4624, 4625.
  * Process Start: 4688 (with CLI arguments).
  * Object Access: 4663.
* Sysmon Logic: Reference for specific Event IDs like ID 1 (Process Create), ID 3 (Network Connection), and ID 22 (DNS Query).
