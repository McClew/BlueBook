# Windows Event Logs

## Key Log Channels

* **Security Log:** The primary source for authentication, authorization, and audit events.
* **System Log:** For OS component and driver issues.
* **Application Log:** Events from installed software.
* **Directory Service Log:** On Domain Controllers, for Active Directory changes.
