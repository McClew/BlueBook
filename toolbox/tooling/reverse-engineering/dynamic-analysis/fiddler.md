# Fiddler

{% hint style="info" %}
#### Download & Install

[https://www.telerik.com/download/fiddler](https://www.telerik.com/download/fiddler)
{% endhint %}

***

Fiddler is a powerful web debugging proxy tool used to capture, inspect, and modify HTTP/HTTPS traffic between a user’s computer and the internet. It allows developers, QA engineers, and IT professionals to debug web applications, analyse API traffic, and troubleshoot performance issues.

{% hint style="success" icon="notes-sticky" %}
## Use Case

Ideal for analysing network connections from malware.
{% endhint %}
