# CFF Explorer

{% hint style="info" %}
## Download & Install

[https://ntcore.com/explorer-suite/](https://ntcore.com/explorer-suite/)
{% endhint %}

***

CFF Explorer is a free, advanced Portable Executable (PE) editor designed to allow the inspection, modification, and analysis of executable files (EXE, DLL). It provides tooling for editing PE headers, sections, imports, and resources, including full support for .NET internal structures and 64-bit files.

This tool provides some buit in unpacking functionality.
