# HxD

{% hint style="info" %}
## Download & Install

[https://mh-nexus.de/en/hxd/](https://mh-nexus.de/en/hxd/)
{% endhint %}

***

HxD is a carefully designed and fast hex editor which, additionally to raw disk editing and modifying of main memory (RAM), handles files of any size.

{% hint style="success" icon="notes-sticky" %}
## Use Case

Useful for inspecting file structures, identifying file signatures (magic bytes), and extracting embedded strings.
{% endhint %}
