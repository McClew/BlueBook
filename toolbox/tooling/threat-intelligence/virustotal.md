# VirusTotal

VirusTotal is a free, Google-owned online service that analyses suspicious files, domains, URLs, and IP addresses for malware using over 70 different antivirus engines and blocklisting services. It serves as a rapid, crowdsourced threat detection tool to identify viruses, worms, and trojans.
